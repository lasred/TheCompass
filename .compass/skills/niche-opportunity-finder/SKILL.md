## Analysis Framework

For each opportunity, provide:

### 1. Business Profile
- What they do
- Typical company size (CITE SOURCE or mark ESTIMATE)
- Revenue ranges (CITE SOURCE or mark ESTIMATE)
- How many exist (CITE SOURCE or mark ESTIMATE)

### 2. Pain Points (MUST HAVE EVIDENCE)
For each pain point:
- **Description:** What's the problem?
- **Evidence:** [Link to Reddit], [Link to forum], [Link to review]
- **Impact:** Time/money cost (CITE SOURCE or mark ESTIMATE)
- **Pattern:** Must show 2-3 sources experiencing this

### 3. Existing Competition (MUST VERIFY)
- List actual competitors found
- Link to pricing pages
- Note what they're missing (based on reviews)
- If claiming "no good solution," show search results proving it

### 4. Willingness to Pay (SHOW CALCULATIONS)
- Current cost of problem (CITE SOURCE or ESTIMATE with reasoning)
- ROI calculation (show the math)
- Budget range (explain reasoning)
- Clearly label estimates

### 5. Where to Find Them (VERIFY EXIST)
- Associations: [Name] ([URL]) - [member count if available]
- Directories: [Name] ([URL])
- Groups: [Name] ([URL]) - [member count]
- Conferences: [Name] ([URL])
