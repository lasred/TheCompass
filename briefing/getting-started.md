# Getting Started with The Compass

## Step 1: Install Requirements

**You need:**
- VS Code
- Claude Code extension for VS Code
- Anthropic API key
- Git

**Don't have Claude Code?** Get it at: https://claude.ai/download

## Step 2: Clone The Compass

```bash
git clone [repo-url]
cd the-compass
```

## Step 3: Open in VS Code

```bash
code .
```

Make sure Claude Code extension is active.

## Step 4: Configure Your API Key

The Compass uses your Anthropic API to run.

Add your API key to Claude Code settings (it will prompt you if needed).

## Step 5: Start The Compass

Open Claude Code in VS Code.

Say "hello" to The Compass.

The Compass will:
1. Check for user-config.json
2. Ask you questions (name, goal, why)
3. Create your config file
4. Welcome you to Mission 1

## Step 6: Begin Mission 1

The Compass will tell you:
- What Mission 1 is about
- What you're doing this week
- How to use the skills
- Where to find resources

Follow its guidance.

## Tips for Success

**Work with The Compass daily:**
- Even 30 minutes a day keeps momentum

**Use the skills:**
- Don't reinvent the wheel - let the skills accelerate you

**Go back to camp frequently:**
- Share progress: https://www.skool.com/base-camp
- Get feedback
- Stay connected

**Don't wait for perfect:**
- 80% done is good enough
- Ship rough, iterate later

**Join live calls:**
- Every Wednesday 2pm PT
- Zoom link in Basecamp

## Common Questions

**Q: Do I need coding experience?**
A: No. Claude Code builds the software. You identify the problems and close the clients.

**Q: How much time does this take?**
A: Plan for 5-10 hours/week. More if you want to move faster.

**Q: What if I get stuck?**
A: Head back to camp and post your question. The community helps. Zane helps on live calls.

**Q: Can I change my niche later?**
A: Yes. Mission 1 is flexible. You can pivot.

**Q: When do I start talking to clients?**
A: Mission 3 (Week 3). First two weeks are research only.
